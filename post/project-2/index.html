Project II: Analysis of the Most Popular Workout Channels (YouTube Data API)

April 13, 2022

In this project, I made use of the YouTube Data API to pull data about the most popular YouTube workout channels, and conducted data analysis to extract insights about people’s preferences and interests when it comes to online workout videos.

Languages & Tools: Python(packages: pandas, seaborn), YouTube Data API

Skills: web scraping, data collection, data cleaning, data analysis

Github link: Popular_workout_channels_analysis_YouTube_API

Viz:

Discoveries:

This explorative data analysis project of the most popular Youtube workout channels, based on data collected by calling YouTube Data API, are able to identify the following trends:

  • ‘Abs’ workout videos have the highest potential of becoming immensely popular.
  • Workout videos that promise fast results and that are short in length have a better chance of getting a high number of views.
