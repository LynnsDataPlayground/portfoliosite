Project III: Ecommerce Store Web Traffic Analysis (SQL)

April 12, 2022

In this project, I analyzed web traffic data for 5 months (Oct 2019 – Feb 2020) from a medium cosmetics online store. Each row in the dataset represents an event. All events are related to products and users. Each event is like many-to-many relation between products and users.

Language & Tools: SQL

Skills: Data cleaning, data wrangling, data Analysis

GitHub Link: Ecommerce Store Web Traffic SQL Project

KPIs Calculated:

  • Session KPIs: ession with product views, session with add-to-cart, session with remove-from-cart and session with check-out;

  • Add_to_cart rate, cart_abandonment rate, per-session value, conversation rate;

  • Repeat customer revenue, repeat purchase rate, one-time customer revenue, etc.

Data Source:

Data was collected by Open CDP project, and made available by Kaggle user Michael Kechinov and REES46 Marketing Platform.
